# What's this?
A workspace for my "Board Game Club Integration".

# Getting Started

```
git clone {repository url}
```

install python(v2.7 or above)

install node.js(npm is required)

```
cd learnjs
```

```
npm install
```

```
npm run build
```

```
./sspa server 
(local server will stand up)
```
